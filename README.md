# Site

Battlefield Japan Community 公式ホームページ

## 開発
### 基本的な開発
pull requestを送ってください。  
レビュワーを必ず立ててください。
### StatusCheck
pull requestのMainBrunchに対するCodeQLが通されます。  
完了と出るまではマージをすることができません。
### マージ
マージは絶対的なSquashMergeになります。
### 質問
質問はDiscussion・Issueでお願いします。

## 実装
### フロントエンド
